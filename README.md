# vulu Launcher

Bu launcher'ı aslen arkadaşlarımla modlu minecraft oynamak için yapıyordum, curseforge profilleri, mods klasörü, cart curt derken iş epey karışıyordu bende mod paketlerini direk profil halinde gönderebileceğim bir uygulama istedim ve ortaya böyle bir şey çıktı şu anda epey eksiği var ancak zamanla eklenecek ve geliştirilecek.
